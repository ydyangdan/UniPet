# Roadmap

UniPet is a universal visual status layer for AI coding agents. The roadmap
keeps the runtime lightweight while making the project easier to try, extend,
and trust.

## v0.1.x: Trust and Onboarding

- Keep `npm install -g uni-pet` as the fastest developer path.
- Add CI, issue templates, contribution docs, and security policy.
- Improve `unipet doctor` and `unipet demo`.
- Document custom agent integration with CLI, HTTP, and WebSocket examples.

## v0.2.x: Better Companion Behavior

- Continue improving idle moments, state transitions, and bubble timing.
- Keep the bridge protocol stable and derive animation locally.
- Improve drag, hover, click, and quiet-mode behavior.

## v0.3.x: Pet Ecosystem

- Improve local pet import and validation commands.
- Document the Codex-compatible pet format.
- Harden pet archive validation for size, paths, and symlinks.

## Later

- Optional MCP server for agents that prefer tool-based integration.
- GitHub Release desktop packages for users who do not want npm.
- Lightweight runtime experiments if Electron becomes the main bottleneck.

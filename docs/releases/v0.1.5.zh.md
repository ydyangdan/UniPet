# UniPet v0.1.5

这一版重点是 Agent 状态可视化和桌面陪伴体验打磨。

## 亮点

- 对齐 Codex 风格宠物 manifest 的渲染方式，包括帧时长、循环起点、fallback
  动画和稳定的时间计算。
- 新增轻量状态卡片，可在宠物旁边查看 Agent 来源、展示状态、任务摘要和持续时间。
- 在不改变桥接协议的前提下，整理展示语义：运行中、思考中、等待中、需要确认、完成、遇到问题。
- 优化气泡文案，让表达更有陪伴感，同时仍然贴合 Agent 正在做的真实动作。
- 安装 Codex Pet Share 宠物时保留内嵌 manifest，并加强自定义宠物校验。
- 更新 README 演示 GIF，更完整展示状态流转。
- 新增 overlay smoke 检查，覆盖本地 bridge 和核心状态演示流程。

## 安装

```bash
npm install -g uni-pet
```

## 更新

```bash
npm update -g uni-pet
```

## 快速体验

```bash
unipet start
unipet demo
unipet doctor
unipet agent status
```

## 连接 Agent

```bash
unipet agent add codex
unipet agent add claude-code
unipet agent add hermes
unipet agent add openclaw
unipet agent add deepseek-tui
```

## 已验证

- npm 包：`uni-pet@0.1.5`
- 桥接协议保持稳定：`source`、`state`、`message`、`action`、`ttl`
- 运行时：Node.js + Electron，本地优先，只监听 localhost

# UniPet v0.1.4

Trust, onboarding, and companion polish.

## Highlights

- Added CI checks across Windows, macOS, and Linux.
- Improved `unipet doctor` and `unipet agent status` with actionable `next:`
  guidance.
- Added `unipet demo` for a quick local status visualization walkthrough.
- Improved companion behavior, idle moments, hover and drag feedback, and
  bubble timing while keeping the bridge protocol stable.
- Added `unipet pet validate` and `unipet pet import` for local pet authors.
- Added npm package smoke-install verification for release confidence.

## Install

```bash
npm install -g uni-pet
```

## Update

```bash
npm update -g uni-pet
```

## Try It

```bash
unipet start
unipet demo
unipet doctor
unipet agent status
```

## Connect An Agent

```bash
unipet agent add codex
unipet agent add claude-code
unipet agent add hermes
unipet agent add openclaw
unipet agent add deepseek-tui
```

## Verified

- npm package: `uni-pet@0.1.4`
- CI: Windows, macOS, Linux
- Bridge protocol: `source`, `state`, `message`, `action`, `ttl`
- Runtime: Node.js + Electron, local-first localhost bridge

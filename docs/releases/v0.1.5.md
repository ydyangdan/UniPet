# UniPet v0.1.5

Agent state visualization and companion polish.

## Highlights

- Aligned spritesheet playback with Codex-style pet manifests, including frame
  durations, loop starts, fallback rows, and deterministic timing.
- Added a compact poster-style status card beside the pet for Agent source,
  display status, task summary, and elapsed time.
- Improved status semantics without changing the bridge protocol: running,
  thinking, waiting, needs confirmation, done, and problem states are now clearer
  in the overlay.
- Reworked bubble copy so companion text stays cute but grounded in what the
  Agent is actually doing.
- Preserved embedded Codex Pet Share manifests during install and hardened
  custom pet validation.
- Refreshed the README demo GIF to show more of the state flow.
- Added overlay smoke coverage for the local bridge and core state walkthrough.

## Install

```bash
npm install -g uni-pet
```

## Update

```bash
npm update -g uni-pet
```

## Try It

```bash
unipet start
unipet demo
unipet doctor
unipet agent status
```

## Connect An Agent

```bash
unipet agent add codex
unipet agent add claude-code
unipet agent add hermes
unipet agent add openclaw
unipet agent add deepseek-tui
```

## Verified

- npm package: `uni-pet@0.1.5`
- Bridge protocol remains stable: `source`, `state`, `message`, `action`, `ttl`
- Runtime: Node.js + Electron, local-first localhost bridge
- Smoke checks: overlay flow and packaged install

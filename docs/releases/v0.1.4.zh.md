# UniPet v0.1.4

这一版重点是提升可信度、上手体验和桌面伙伴表现。

## 亮点

- 新增 Windows、macOS、Linux 三平台 CI 检查。
- 优化 `unipet doctor` 和 `unipet agent status`，输出更明确的 `next:`
  下一步建议。
- 新增 `unipet demo`，可以快速预览 Agent 状态可视化效果。
- 优化宠物行为、空闲小动作、鼠标悬停/拖拽反馈和气泡时间，同时保持桥接协议稳定。
- 新增 `unipet pet validate` 和 `unipet pet import`，方便本地宠物作者检查和导入宠物。
- 新增 npm 包安装 smoke test，让发布包更可信。

## 安装

```bash
npm install -g uni-pet
```

## 更新

```bash
npm update -g uni-pet
```

## 快速体验

```bash
unipet start
unipet demo
unipet doctor
unipet agent status
```

## 连接 Agent

```bash
unipet agent add codex
unipet agent add claude-code
unipet agent add hermes
unipet agent add openclaw
unipet agent add deepseek-tui
```

## 已验证

- npm 包：`uni-pet@0.1.4`
- CI：Windows、macOS、Linux
- 桥接协议：`source`、`state`、`message`、`action`、`ttl`
- 运行时：Node.js + Electron，本地优先，只监听 localhost
